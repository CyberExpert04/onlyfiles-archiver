# onlyfiles-archiver

This program takes an ```onlyfiles.txt``` input and parses the links inside to back up the referenced data. 

You can find the most updated version of this file at ```pillows.su/onlyfiles.txt```.

Right click on the page, select Save As, then save the file as a TXT file.

Move the saved file to the working directory of this archiver.

Open the directory in terminal and run ```python onlyfiles-archiver.py```.
